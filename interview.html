Ques 4. How may types of headings are there in HTML? And how to decide which to use where?

Ans.A HTML heading tag is used to define the headings of a page. There are six levels of headings defined by HTML. These 6 heading elements are h1, h2, h3, h4, h5, and h6; with h1 being the highest level and h6 being the least.
h1 is used for main heading. (Biggest in size)
h2 is used for subheadings
if there are further sections under the subheadings then the h3 elements is used.
h6 for the small heading (smallest one).
Browsers display the contents of headings in different sizes. The exact size at which each browser shows the heading can vary slightly. Users can also adjust the size of text in their browser.


Ques 5. Diferrence between block and inline

Ans.
Block elements: They consume the entire width available irrespective of their sufficiency. They always start in a new line and have top and bottom margins. It does not contain any other elements next to it.
Example:
div, ul, ol, li
Inline elements:Inline elements occupy only enough width that is sufficient to it and allows other elements next to it which are inline. Inline elements do not start from a new line and do not have top and bottom margins as block elements have.
Example:
a, br, img, input

Ques 10.What are semantic tags?

Ans. The HTML semantics refers to the tags that provide meaning to an HTML page rather than just presentation. It makes HTML more comprehensible by better defining the different sections and layout of web pages. The semantic HTML tags help the search engines and other user devices to determine the importance and context of web pages.
The pages made with semantic elements are much easier to read.
Examples:
header, nav, section, article.
